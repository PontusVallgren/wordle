Project Wordle

In this project i tried to create a similar game to wordle, with focus on learning React. What this project includes:

  • - Game GUI created by using React.
  • - Node/Express server with REST API and handling game logic.
  • - Server written with typescript.
  • - Mongodb as database for storing users highscore.
  • - Highscore list server rendered with help of EJS.
  • - Includes tests for game algorithms.
